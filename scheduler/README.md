# PrimeUI Pro Scheduler Examples

Quickstarts for building scheduling applications with PrimeUI Pro Scheduler.

[Read the Vue Scheduler documentation](https://vue.primeuipro.dev/scheduler).

## Availability

There are no runnable Scheduler quickstarts in this repository yet.

The planned Vue applications are:

- `vue/vite-quickstart`
- `vue/nuxt-quickstart`
- `vue/vite-tailwind-quickstart`
- `vue/nuxt-tailwind-quickstart`

React and Angular quickstarts are also planned. Their directories are placeholders until runnable examples are added.
