# PrimeUI Pro Charts Examples

Quickstarts for building data visualizations with PrimeUI Pro Charts.

[Read the Vue Charts documentation](https://vue.primeuipro.dev/charts).

## Availability

There are no runnable Charts quickstarts in this repository yet.

The planned Vue applications are:

- `vue/vite-quickstart`
- `vue/nuxt-quickstart`
- `vue/vite-tailwind-quickstart`
- `vue/nuxt-tailwind-quickstart`

React and Angular quickstarts are also planned. Their directories are placeholders until runnable examples are added.
