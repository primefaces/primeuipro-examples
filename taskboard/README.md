# PrimeUI Pro TaskBoard Examples

Quickstarts for building kanban workflows with PrimeUI Pro TaskBoard.

[Read the Vue TaskBoard documentation](https://vue.primeuipro.dev/taskboard).

## Available Quickstarts

- [Vite quickstart](./vue/vite-quickstart) - PrimeOne UI parts, PrimeVue controls, a standard board, and task CRUD
- [Nuxt quickstart](./vue/nuxt-quickstart) - PrimeOne UI parts, PrimeVue controls, a standard board, and task CRUD
- [Vite + Tailwind quickstart](./vue/vite-tailwind-quickstart) - Tailwind UI parts, native controls, a standard board, and task CRUD without PrimeVue
- [Nuxt + Tailwind quickstart](./vue/nuxt-tailwind-quickstart) - Tailwind UI parts, native controls, a standard board, and task CRUD without PrimeVue

## Planned Quickstarts

- `react` using [PrimeReact NextChapter](https://github.com/primefaces/primereact-nextchapter)
- `angular` using [PrimeNG NextChapter](https://github.com/primefaces/primeng-nextchapter)
