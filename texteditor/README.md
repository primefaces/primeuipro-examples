# PrimeUI Pro TextEditor Examples

Quickstarts for building rich text editing experiences with PrimeUI Pro TextEditor.

[Read the Vue TextEditor documentation](https://vue.primeuipro.dev/texteditor).

## Availability

There are no runnable TextEditor quickstarts in this repository yet.

The planned Vue applications are:

- `vue/vite-quickstart`
- `vue/nuxt-quickstart`
- `vue/vite-tailwind-quickstart`
- `vue/nuxt-tailwind-quickstart`

React quickstarts will use [PrimeReact](https://primereact.org/). Angular quickstarts will use [PrimeNG](https://primeng.org/). Their directories are placeholders until runnable examples are added.
