# PrimeUI Pro TextEditor Examples

Quickstarts for building rich text editing experiences with PrimeUI Pro TextEditor.

[Read the Vue TextEditor documentation](https://vue.primeuipro.dev/texteditor).

## Vue Quickstarts

- `vue/vite-quickstart`
- `vue/nuxt-quickstart`
- `vue/vite-tailwind-quickstart`
- `vue/nuxt-tailwind-quickstart`

The default Vite and Nuxt quickstarts use PrimeOne UI parts with PrimeVue. The Tailwind variants use editable Tailwind UI parts and do not depend on PrimeVue.

React quickstarts will use [PrimeReact](https://primereact.org/). Angular quickstarts will use [PrimeNG](https://primeng.org/). Their directories remain placeholders until runnable examples are added.
